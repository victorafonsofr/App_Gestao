# 👥 Sistema de Gestão de Empregados (Simple Employee Management)

Este é um sistema de console simples desenvolvido em Java para demonstrar a aplicação prática de conceitos fundamentais da Programação Orientada a Objetos (POO), especificamente **Herança** e **Polimorfismo**. O projeto foca nas funcionalidades essenciais de **Cadastro** e **Visualização** de diferentes tipos de funcionários.

---

## 🚀 Funcionalidades

* **Cadastrar Empregados:** Permite a inserção de funcionários comuns e categorias específicas.
* **Visualizar Dados:** Lista todos os empregados cadastrados, exibindo o cálculo correto de seus respectivos salários de forma dinâmica.

---

## 🧬 Conceitos de POO Aplicados

### 1. Herança
A classe base `Empregado` encapsula os atributos comuns (Nome, cpf, endereço). Classes específicas como `EmpregadoCLT` e `EmpregadoHorista` herdam de `Empregado`, reaproveitando o código e adicionando particularidades.

### 2. Polimorfismo
métodos castados para utilizar de métodos exclusivos de cada classe

---

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** Java (versão 25)
* **Paradigma:** Programação Orientada a Objetos (POO)
